笔记本：游戏本冲规模，轻薄本以内容提效

游戏本

导购型规模最高：9,834 万元 GMV、4.15% 转化；内容型 GMV 4,568 万元，但支付 GMV 渗透更高（48.0%），显示其对应商品在店内成交中的带动更深。

建议：大促主推拯救者时，用值得买做资源位和爆料拉峰；用荣牛哥 / 哥登做测评、性能对比与长尾搜索承接，避免把内容预算全压在同一天。

轻薄本

内容型转化最高（2.58%），高于社群型（1.23%）与导购型（1.67%）。社群型虽覆盖 11.7 万访客，但成交效率偏弱，适合新品信息触达与权益提醒，而非单独背销量目标。

建议：小新 / YOGA 等轻薄本先以内容种草解释性能、续航、便携等差异，再由社群承接优惠；按内容收藏/点击与社群进店分别考核。

下一轮投放的三条执行规则

  1. 拯救者平板：给值得买独立爆发预算与首焦/补贴资源，采用 D-1 预热、D0 爆发、D+1 返场；Y900 与 Y700 分商品设库存阈值。
  2. 小新平板与轻薄本：建立内容—社群接力包。内容发布后 24 小时内由社群发券承接，并以“内容点击 → 进店 → 支付”的链路 KPI 结算。
  3. 游戏本：导购型保障峰值 GMV，内容型负责性能测评和长尾转化；分别考核爆发 GMV 与支付渗透，避免用单一 CPA 误伤内容价值。

口径与数据质量说明

归因口径：CPS 明细按商品 ID 与《淘客-团长-ID 对应表》关联；笔记本“值得买和荣牛”1 个联合标注商品保留为组合归因，不拆分给单一团长。

核心指标：支付 GMV = 付款金额；进店人数 = CPS 明细进店人数；转化率 = 付款件数 / 进店人数；支付渗透 = 付款金额 / 总支付金额；ROI = 付款金额 / 付款支出费用。各指标仅反映已映射商品与可用日期。
